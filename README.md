# Consultorio Luz y Vida - Sistema de Gestión de Citas Médicas

Este es un sistema para la gestión de citas médicas para el consultorio **Luz y Vida**. El sistema permite a los pacientes crear, ver y actualizar citas médicas de manera sencilla y eficiente.

## Propósito

El objetivo de este proyecto es proporcionar una solución digital para la gestión de citas médicas, mejorando la eficiencia y reduciendo el riesgo de errores. Este sistema será utilizado por los pacientes para agendar citas médicas y consultar su historial de citas.
